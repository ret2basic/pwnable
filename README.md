# pwnable

"pwnable" is my personal pwn challenge collection. Challenges are classified by techniques used:

- **ROP**
  - ret2syscall
  - ret2libc
  - ret2csu
  - BROP
  - SROP
  - stack pivoting
  - ret2dl-resolve
- **Heap**
  - UAF
  - fastbin
  - house of spirit
  - unsafe unlink
  - off-by-one
  - house of einherjar
  - overlapping chunks
  - house of force
  - unsorted bin
  - large bin
